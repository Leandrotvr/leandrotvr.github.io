Proyecto de Lista de Tareas

Este proyecto demuestra mis habilidades en el desarrollo web full-stack, utilizando las siguientes tecnologías:

  • **Frontend:** HTML, CSS y JavaScript (Vanilla JS).
  • **Backend:** Node.js y Express.js para la API REST.
  • **Base de Datos:** MongoDB para el almacenamiento de datos.
  • **Despliegue:** Render para el despliegue del servicio web.
  • **Control de Versiones:** Git y GitHub para la colaboración y el control de versiones.
